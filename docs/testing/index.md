# Testing Documentation

This section contains comprehensive documentation about testing, coverage, and quality standards for the starter-api-spring-mysql project.

## Overview

The project enforces **80% minimum code coverage** for line, branch, and instruction coverage. All documentation related to testing practices, coverage reports, and test organization can be found here.

## Key Testing Standards

- **Minimum Coverage**: 80% for lines, branches, and instructions
- **Testing Framework**: JUnit 5 with Mockito
- **Coverage Tool**: JaCoCo 0.8.14
- **Build Command**: `mvn verify`
